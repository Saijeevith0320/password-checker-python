# Password Strength Checker & Breach Detector (Python)

A Python tool that analyzes password strength, checks for breach patterns, calculates entropy, and generates a CSV report.

## Features
- Strength analysis
- Breached pattern detection
- Entropy calculation
- CSV report generation
- Simple CLI interface

## Usage
```
python password_checker.py --password MyPassword123! --output report.csv
```
